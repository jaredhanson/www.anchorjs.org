Anchor

About Anchor

Anchor is a modular platform for writing browser-based JavaScript applications. Anchor provides a concise, consistent set of core libraries for DOM traversal, Ajax requests, HTML5 APIs, and more. Anchor is designed to change the way you write JavaScript, again.
